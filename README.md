🔐 ZERYNX Hash Generator

A simple and lightweight Python tool for generating cryptographic hashes from any text.

Features

* Generate hashes instantly
* Supports multiple algorithms:
    * MD5
    * SHA1
    * SHA224
    * SHA256
    * SHA384
    * SHA512
* Easy to use
* Built with Python
* Open Source

Installation

~~~
git clone https://github.com/YOUR_USERNAME/ZERYNX-Hash-Generator.git
cd ZERYNX-Hash-Generator
python main.py
~~~
Example

Input

Hello ZERYNX

Output

9e5d...

* Python 3.x
